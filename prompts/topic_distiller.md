## Принципы отбора (по убыванию приоритета)

### 1. Hook = конкретный нарратив / боль / результат, не метафора
Первая строка темы/поста должна формулировать **конкретную проблему, боль или
измеримый результат** (нарратив), а не мифологию/абстракцию.

**Данные (Module D, 2026-05-29, IG n=13, high confidence):**
- Pain-point хуки → reach **161** («50% запросов в проде падают»), **148** («Платить за AI-инструменты в 2026 — странно»)
- Нарратив/результат хуки → reach **>100** («43 провала. Потом 250 000 звёзд на GitHub за 2 месяца»)
- Мифология/метафора → reach **10** («Гефест был хромым богом»), **58** («Один отдал глаз за мудрость»)
- Вывод: конкретика залетает в 3-16× лучше (нарратив/боль/результат reach 167 vs абстракция 0). Алгоритм IG пушит actionable-контент.

### 2. Приоритетные категории (кросс-канальный резонанс)
Темы, которые заходят сразу на YouTube + Plausible + Instagram:
- `ai-agents` (YT до 1396 views, топ-4 Plausible, IG, YT)
- `infra` / serverless / API reliability
- `llm-tools` / how-to / production patterns

**Целевая доля тем про `ai-agents` и `llm-tools` — ≥98% от еженедельного контента во всех каналах.**

### 3. Доля форматов
- Pain-point / how-to / production-проблемы: **целевая доля ≥84%**
- ai-agents / llm-tools: **+40%** к текущей доле (за счёт снижения generic/other; кросс-канальный резонанс: топ-4 Plausible, IG, YT)
- Generic/other: **−40%** от текущей доли
- Мифология / метафора / «красивые» нарративы: **≤10%** (низкий reach, для брендинга не для роста)
- Сценарии с низким перформансом (median reach < 100): **уменьшить объём на 50%**, фокус на качестве hook (см. правило №1) — рост важнее объёма

### 4. Назначение формата Волкова (`volkov_format`) — НОВОЕ 16.06
Каждой теме проставлять поле `topics.volkov_format` (метод Волкова, см. `VOLKOV_METHOD.md`):

| volkov_format | Когда выбирать | Сигнал |
|---|---|---|
| `provocation` | свежий горячий инфоповод + угроза профессии/бизнесу (по умолчанию для B2B) | комменты |
| `resource_list` | подборка инструментов/ресурсов/чек-лист | репосты+сохранения |
| `7_signs` | диагностика читателя («признаки X», «как отбираю») | сохранения+комменты |

Правило по умолчанию: горячий тренд (high hype_score, свежий) → `provocation`;
«N инструментов/ресурсов» → `resource_list`; «признаки/чек-лист/как понять» → `7_signs`.
Если непонятно → `provocation` (дефолт для DACH B2B).
Ротация недели: ПН провокация / СР демо-видео / ПТ список (не лить провокацию каждый день — выжигает).
